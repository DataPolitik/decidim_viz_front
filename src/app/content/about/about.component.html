¿Qué es Decidim.vis?

Decidim.vis es una prueba de concepto de un visor de información de una instancia de Decidim. El objetivo de Decidim.vis es atender a la necesidad de la Comunidad de usuarios de Decidim de conseguir, rapidamente, métricas e información estadística que les pueda ser de utilidad.

¿De dónde saca los datos Decidim.vis?

Decidim.vis obtiene los datos de dos fuentes:

  • API de de la instancia de Decidim.
  • Ficheros de descarga masiva de Decidim.

Actualmente, para la prueba de concepto, utilizamos la instancia de Futureu como fuente de datos.
